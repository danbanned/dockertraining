# BrightPath Tutoring App Features

## Core Features
1. Secure student and parent login with role-based access.
2. Tutor matching by subject, grade level, and preferred schedule.
3. Lesson catalog where students can browse and enroll in sessions.
4. Progress dashboard showing attendance, assignment completion, and skill growth.
5. Session booking and calendar reminders for upcoming tutoring appointments.
6. Messaging between families and tutors for follow-up and support.

## System Needs
1. Reliable database connectivity to store users, schedules, and learning progress.
2. Authentication and authorization controls for student, parent, tutor, and admin roles.
3. API layer for scheduling, lesson retrieval, and analytics data.
4. Observability/logging for troubleshooting session and booking issues.
